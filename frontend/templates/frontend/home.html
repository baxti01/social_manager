О проекте

 Это сервис автоматизации публикации для социальных сетей. Сервис написан на основе API и имеет пользовательский интерфейс в виде сайта.
 Сервис написан на языке программирования Python с использованием его фреймворков Django и Django Rest Framework.
 Для использования сервиса нужно пройти регистрацию и подключить хотя бы одну из социальных сетей, сейчас доступно подключение Telegram и Instagram.
